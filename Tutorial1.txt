## A learning Tutorial of basic program(Hello World!) in all Programming languages.

C : functional oriented 

#include <stdio.h>
int main() {
   // printf() displays the string inside quotation
   printf("Hello, World!");
   return 0;
}

Java: object oriented 

class HelloWorld {
    public static void main(String args[]) {
        System.out.println("Hello World!");
    }
}


python3:

print("Hello, World!")
